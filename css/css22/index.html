概要

この仕様書はカスケーディングスタイルシート (CSS) レベル 2 を定義します。CSSは、著者やユーザーがスタイル（例: フォントや間隔）を構造化文書（例: HTML文書やXMLアプリケーション）に付与することを可能にするスタイルシート言語です。文書のプレゼンテーションスタイルを文書の内容から分離することで、CSSはウェブ作成やサイト管理を簡素化します。

CSS 2.2はCSSレベル 2の[CSS2]の2回目の改訂版です。これはCSS 2.1（CSSレベル 2の最初の改訂版）のいくつかのエラーを修正しています。これは最新のCSSバージョンではありません。「CSS Snapshot」[CSS]を参照してください。CSS全体を構成する仕様書の概要が記されています。

CSSレベル 2はメディア固有のスタイルシートをサポートしており、著者が文書のプレゼンテーションを視覚ブラウザ、音響デバイス、プリンター、点字デバイス、ハンドヘルドデバイスなどに対応させることができます。また、コンテンツの位置指定、表レイアウト、国際化のための機能、ユーザーインターフェースに関連するいくつかのプロパティもサポートしています。
